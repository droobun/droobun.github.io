Summary: Latitude vs. X


The goal of this project was to analyze how weather changes based on how close a city is to the equator. To figure this out, we pulled data from the OpenWeatherMap Api to create a dataset around 500 different cities.

After structuring the data set, we used Matplotlib to plot various coorelations between weather and latitude. Factors we looked at were: temperature, cloudiness, wind speed, and humidity. This site allows anyone to research and consume the source data and visualizations that were created as a part of the project.
